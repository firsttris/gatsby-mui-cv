# Erstellung von Benutzeroberflächen im Bereich der Telekommunikationsüberwachung
10/2022 - 06/2023 Bad Homburg/DE   
ATIS® systems über Solcom   


Unterstützen und schulen der ATIS systems beim erstellen von Benutzeroberflächen mit React.

## Ergebnisse
- Bootstrap React/TypeScript-Projekt mit NX-Tools
- Cypress-Tests erstellt, die API-Antworten automatisch aufzeichnen, um sie im nächsten Lauf als Mocks zu verwenden
- Webpack so konfigurieren, dass es Anfragen für mehrere APIs weiterleitet
- TypeScript-Codegenerierung vom OpenAPI-Endpunkt einrichten
- Customizing von MUI Componenten
- Umsetzung von Features nach Anforderung des Kunden
- Jenkins-Pipeline aufgesetzt
