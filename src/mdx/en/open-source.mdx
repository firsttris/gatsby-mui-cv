## VS Code Addon Jest Runner
#### 12/2017 - today | [Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=firsttris.vscode-jest-runner) | [Github Repository](https://github.com/firsttris/vscode-jest-runner.git)

Jest Runner provides an easy way to run or debug a specific test as is possible in IntelliJ/Webstorm. 
[vscode-jest-runner](https://marketplace.visualstudio.com/items?itemName=firsttris.vscode-jest-runner) focuses on running or debugging a specific test or test suite, 
while [vscode -jest](https://marketplace.visualstudio.com/items?itemName=Orta.vscode-jest) runs your current test suite every time you change it.

- #2 if you search for Jest in the Visual Studio Marketplace
- Jest-Runner is installed as a recommended extension from [nrwl's NX-Tools](https://github.com/nrwl/nx/issues/3868).
- 896,604 installs
- 5-star rating in the Visual Studio Marketplace
- 231 stars on Github, 3 contributors, 38 contributors

## SendToKodi
#### 01/2016 - today | [Github Repository](https://github.com/firsttris/plugin.video.sendtokodi)

SendToKodi is a Kodi addon (Python) that resolves various video streams with youtube-dl to play them on Kodi

- A Kodi addon (Python) that extracts video streams from website urls using YoutubeDL
- A Github Actions job that regularly rebuilds the Kodi addon with an up-to-date YoutubeDL version
- A native iOS app (Swift) that provides the built-in sharing option to send links to Kodi
- Added in-app payment option for multiple hosts
- The in-app payment makes about $15 per quarter, which is enough to pay the developer fee
- Official Kodi App Kore supports SendToKodi
- Chrome addon that allows to send video links to Kodi
- 105 stars, 2 employees, 11 contributors

## React-Track
#### 01/2018 - heute | [Github Repository](https://github.com/firsttris/react-track)
A simple timetracking tool with reactjs and lowdb.    
A customer of mine was looking for a very simple timetracking application which runs in the web and is responsive on mobile. His idea was that every employee could book and check their times with their mobile device.

Since we couldn't find a software with modern technology and a effortable price we created our own tool.

- Responsive Web Design
- Crossplattform compatibility
- User Management
- Time Tracking reports
- 2 contributors
- used by 5 small businesses todo their timetracking
