## VS Code Addon Jest Runner
#### 12/2017 - heute | [Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=firsttris.vscode-jest-runner) | [Github Repository](https://github.com/firsttris/vscode-jest-runner.git)

Jest Runner bietet eine einfache Möglichkeit, einen bestimmten Test auszuführen oder zu debuggen, wie es in IntelliJ/Webstorm möglich ist. 
[vscode-jest-runner](https://marketplace.visualstudio.com/items?itemName=firsttris.vscode-jest-runner) konzentriert sich dabei auf die Ausführung oder das Debuggen eines bestimmten Tests oder einer bestimmten Testsuite, 
während [vscode-jest](https://marketplace.visualstudio.com/items?itemName=Orta.vscode-jest) Ihre aktuelle Testsuite jedes Mal ausführt, wenn Sie sie ändern.

- #2, wenn Sie im Visual Studio Marketplace nach Jest suchen
- Jest-Runner wird als empfohlene Erweiterung von [nrwl's NX-Tools](https://github.com/nrwl/nx/issues/3868) installiert
- 896,604 Installationen
- 5-Sterne-Bewertung im Visual Studio Marketplace
- 231 Sterne auf Github, 3 Mitarbeiter, 38 Mitwirkende

## SendToKodi
#### 01/2016 - heute | [Github Repository](https://github.com/firsttris/plugin.video.sendtokodi)

SendToKodi ist ein Kodi Addon (Python), das verschiedene Videostreams mit youtube-dl auflöst, um sie auf Kodi abzuspielen

- Ein Kodi-Addon (Python), das Videostreams von Website-URLs mit YoutubeDL extrahiert
- Ein Github Actions Job, der das Kodi-Addon regelmäßig mit einer aktuellen YoutubeDL-Version neu erstellt
- Eine native iOS-App (Swift), die die integrierte Freigabeoption zum Senden von Links an Kodi bereitstellt
- In-App-Zahlungs Option für mehrere Hosts hinzugefügt
- Mit der In-App-Zahlung werden ungefähr 15 US-Dollar pro Quartal erzielt, was ausreicht, um die Entwicklergebühr zu zahlen
- Offizielle Kodi App Kore unterstützt SendToKodi
- Chrome Addon das ermöglicht Video links an Kodi zu senden
- 105 Sterne, 2 Mitarbeiter, 11 Mitwirkende

## React-Track
#### 01/2018 - heute | [Github Repository](https://github.com/firsttris/react-track)
A simple timetracking tool with reactjs and lowdb.    
A customer of mine was looking for a very simple timetracking application which runs in the web and is responsive on mobile. His idea was that every employee could book and check their times with their mobile device.

Since we couldn't find a software with modern technology and a effortable price we created our own tool.

- Backend & Frontend with Apollo GraphQL
- Responsive Web Design
- Crossplattform compatibility
- User Management
- Time Tracking reports
- 2 contributors
- used by 5 small businesses todo their timetracking
